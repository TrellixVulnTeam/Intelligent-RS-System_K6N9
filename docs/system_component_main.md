# 平台主页
本平台主页分为以下四个部分
- 遥感图像微介绍（让用户了解什么是遥感）
- 基本功能介绍及新闻中心（简要介绍各功能，新闻中心提供用户遥感相关讯息）
- 技术实现介绍（介绍平台所用到的开发的技术）
- 登录注册界面（用户需登录后才能使用所有功能）
